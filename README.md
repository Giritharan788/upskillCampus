Internship Project: Deadline Reminder App

This repository contains the internship project and final report submitted for the UpSkillCampus Internship Program.

Project:
Deadline Reminder is a full stack web application that helps users manage tasks and deadlines. Users can add, edit, delete, and complete tasks. Deadlines are sorted based on date and priority.

Technologies Used:
- React
- Tailwind CSS
- Firebase

This project was developed as part of the internship requirements.
